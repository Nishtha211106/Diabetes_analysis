# Diabetes analysis using ML
It is a Machine Learning project that predicts whether a person has diabetes based on medical features like Glucose, Insulin, BMI, Blood Pressure and more.

Dataset: Pima Indians Diabetes Dataset from Kaggle
Concepts Used:
Standard Scaler for feature scaling
Pipeline for clean workflow
Cross Validation for reliable accuracy
GridSearchCV for hyperparameter tuning

Results:
Model Accuracy: 88%
Most important features: Glucose, Insulin, SkinThickness

Insights:
Glucose is the strongest predictor of diabetes (0.50 correlation)
Higher glucose, insulin and skin thickness = higher diabetes risk
Age and DiabetesPedigreeFunction are also moderately correlated with diabetes
